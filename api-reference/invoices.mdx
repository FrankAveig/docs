## Crear Factura

Endpoint para crear una nueva factura electrónica.

### Endpoint

<CodeGroup>```bash POST {endpoint}/v1/m2m/documents/invoice ```</CodeGroup>

### Headers Requeridos

<ParamField header="Authorization" type="string" required>
  Bearer {access_token}
</ParamField>

### Parámetros del Body

<ParamField body="invoice" type="object" required>
  Objeto con la información de la factura
</ParamField>

<ParamField body="unique_id" type="string" required>
  Identificador único del cliente
</ParamField>
