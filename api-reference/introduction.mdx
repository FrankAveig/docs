## Introducción

La API de FacturaYa te permite integrar nuestro sistema de facturación electrónica en tu aplicación. Con nuestra API podrás:

- Generar facturas electrónicas
- Consultar el estado de los documentos
- Gestionar clientes y productos
- Y más...

## Autenticación

Todas las peticiones a la API requieren autenticación mediante un token Bearer. Para obtener el token, consulta la sección de [Autenticación](/api-reference/authentication).
